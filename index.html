Purpose

The purpose of the application is to streamline the goal-setting and reflection process.

Technologies Used

  • Frontend: React.js
  • Backend: Node.js, Express.js
  • Database: MongoDB-Atlas

Key Features

  • Reduced time spent on goal-setting
  • Save your goals in the draft before the final submission
  • Reduced number of meetings
  • Improved visibility of progress through graphs and table
  • Easier collaboration/brainstorming with team

Architecture

Frontend Architecture

The Frontend is built using React.js in combination with Vite

Backend & Database Architecture

The Backend is built using Node.js and Express.js
